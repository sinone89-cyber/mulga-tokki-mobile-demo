물가의 토끼에 오신 걸 환영해요!

🎣 낚시는 자동으로 진행돼요. 가방이 차면 판매!

🎒 가방에서 물고기를 팔아 골드를 벌어요. (상점에서 자동 판매도 해금)

🏪 상점에서 업그레이드·미끼·꾸미기를 구매해요.

🧰 낚시 화면의 미끼 상자에서 미끼를 장착하면 원하는 물고기가 잘 잡혀요.

📖 도감·크라운을 모아 달성 보상으로 🐚조개를 얻고, 조개로 🐾펫을 데려와요.

📋 미션으로 골드·조개를 챙기세요.

💾 진행도는 자동 저장 — 상단 💾로 백업/복원도 할 수 있어요.

❓

도움말 — 게임 가이드

🎣 낚시: 자동으로 진행돼요. 찌가 반응하고 게이지가 차면 한 마리 낚여 가방에 들어갑니다. 손 안 대도 알아서 잡혀요.

📱 조작: 화면 아래 탭(가방·상점·도감·미션·지역)을 눌러 메뉴를 열고, 같은 탭을 다시 누르면 닫혀요.

🎒 가방: 잡은 물고기를 보관해요(칸 인벤토리). 길게 끌어 옮기고, 톡 눌러 회전·선택. 가방이 차면 판매로 골드를 벌고, 상점에서 자동 판매도 해금돼요.

🏪 상점: 골드로 업그레이드(가방 확장·획득량·행운·오프라인 통발 등)·미끼·🐚조개 꾸미기를 사요.

🧰 미끼: 낚시 화면의 미끼 상자를 눌러 장착하면 특정 물고기가 잘 잡혀요(1회당 1개 소모).

📖 도감 · 👑크라운: 물고기를 수집하면 기록돼요. 받을 보상이 있으면 도감 탭에 빨간 점이 떠요.

📋 미션: 일일 미션 + 도전과제를 깨면 골드·🐚조개 보상을 받아요.

🐾 펫: 조개로 펫을 데려오면 화면 아래 바 위를 거닐며 능력치 보너스를 줘요. 재생목록의 🐾 버튼으로 표시/숨김.

🗺️ 지역: 허가증을 사면 새 낚시터(거친 대양·심해·나이트시티·이세계)가 열려요.

🎵 음악: 화면 우측 상단 재생목록에 음원/유튜브 URL·파일을 추가해 들어요. ▸ 버튼으로 접고 펴요.

⏳ 오프라인 통발: 앱을 꺼둔 동안에도 일부를 자동으로 잡아둬요(상점에서 강화, 능동 플레이보다는 적게).

💾 저장: 자동 저장돼요. 상단 💾로 백업 코드를 만들어 다른 기기로 옮길 수 있어요.
