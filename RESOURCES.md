# 리소스 교체 스펙 (세로 모바일판 기준)

기준 캔버스: **800 × 1480** (`body.pt #app`). 1cqh = 14.8px, 1cqw = 8px.
씬 레이어 순서(뒤→앞): **#bg(배경) → .water(낚시터) → #seamfx(오브젝트) → 잔교(dock) → 캐릭터(z3)**.
우선순위: ① 배경 ② 낚시터 ③ 잔교 ④ 오브젝트.

---

## ① 배경 `fishing/bg/bg_r1~5.png`
- **역할**: 화면 상단 띠 — 하늘 / 먼 풍경 / 수평선(지역별).
- **현재**: 1536×1024 (가로형) → 세로에서 `cover`로 좌우·상하 잘림.
- **화면 점유 영역**: `top 0 ~ 40.2cqh` = **800 × 595px** (상단 ~40%).
- **권장 제작 스펙**: **800×600** (또는 2배 1600×1200), 세로 프레이밍. `background:center center/cover`.
  - 이미지 **하단부 = 수평선/먼 물가**(아래 .water와 자연스럽게 이어지게).
  - 중요한 요소는 가운데로(좌우 약간 크롭될 수 있음).

## ② 낚시터(수면) `fishing/bg/water_r1~5.png`
- **역할**: 중하단 물 표면 / 반사 텍스처.
- **현재**: 864×1821 (이미 세로형) — 대체로 OK, 정밀 맞춤 시 재작업.
- **화면 점유 영역**: `top 37.2cqh(550px) ~ 하단탭바 위(1380px)` = **800 × 830px**. `background:center top/cover`.
- **권장 제작 스펙**: **800×850** (2배 1600×1700).
  - 이미지 **상단 끝 = 수면선**(여기서 배경/오브젝트와 만남). 위쪽이 잘리지 않게 상단 정렬(center top).

## ③ 잔교(부두) **[신규, 지역별 5종]** — `fishing/bg/dock_r1~5.png`
- **역할**: 캐릭터가 걸터앉는 나무 데크(현재 임시 SVG).
- **포즈 기준**: 캐릭터(768² 스프라이트, 화면상 폭 ~437px, **중심 x≈204px**)가 **오른쪽 끝에 앉아 오른쪽을 보고 부츠를 늘어뜨림**.
- **화면 점유 영역(현재 SVG)**: `left -16px ~ 272px, top 488px ~ 725px` (폭 ~288px, 높이 ~237px).
  - **데크 상판(앉는 면) ≈ y504px**(캐릭터 좌석 33.7cqh와 일치).
  - **앞 가장자리 ≈ x260px**(부츠가 그 앞으로 달랑).
  - **기둥은 y725px까지**(수면선 550px 아래로 들어감).
- **권장 제작 스펙**: 투명 PNG **~600×400** (2배 1200×800), 가로 데크가 **왼쪽(물가)에서 오른쪽으로** 뻗고 오른쪽 끝에 캐릭터. 받침 기둥 1~2개.
- **지역별 5종** `dock_r1~5.png`. 실루엣/비율은 5종 통일, 재질·테마만 지역별(호수=나무 / 대양=소금기 회색목재 / 심해=어두운목재+발광랜턴 / 시티=네온 금속 / 이세계=발광 크리스탈목재).
- ※ PNG 주시면 임시 SVG(`#pt-dock`)와 교체하고 위치 미세조정.

## ④ 오브젝트(물가 경계) `fishing/bg/seam_r1~5.png`
- **역할**: 수면선의 장식 오브젝트(갈대/바위/연꽃 등 지역별).
- **현재**: 2048×512 (4:1) — 단일 노출, 지역별 상하 보정(`--ptsdy`) 완료. 대체로 OK.
- **화면 점유 영역**: `left 13%(104px), 폭 ~696px, aspect 4:1 → 높이 ~174px`, 수면선(550px)에 걸침.
- **권장 제작 스펙**: 투명 PNG **4:1 유지**(1600×400 또는 현행 2048×512). 좌측에 한 덩어리.
